# Portfolio-grade boundary

This is a curated exhibit from an active passion project. Starforge Canticles is
published as a serial creative pursuit, and the surrounding tooling explores how
that serial can become a narrative video game over time.

Portfolio-grade here means:

- The public Act 1 slice is present.
- Unreleased later-act material stays private until publication.
- Runtime and editor junk are excluded.
- The repo has repeatable validation.
- A reviewer can tell what capability is being demonstrated.
- Future work can happen in public without dragging along workshop state.

The core claim is that creative-scale work benefits from product-grade systems:
data contracts, conversion tools, validation, cleanup rules, release boundaries,
and explicit limitations.
