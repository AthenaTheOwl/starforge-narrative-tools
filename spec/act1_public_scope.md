# Act 1 public scope

This exhibit is scoped to the portion of Starforge Canticles that is already
public on Royal Road.

Included:

- released Act 1 prose files under `prose/act1/`
- tooling used to convert prose into game-ready Ren'Py-style source
- validation checks for release hygiene and generated artifact cleanup
- public-facing notes about the adaptation pipeline

Excluded:

- later-act prose
- future-act outlines
- spoiler-heavy character dossiers
- private workshop task state
- engine-generated runtime files

The goal is to show a repeatable creative-production workflow without exposing
unpublished story material.
